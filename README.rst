=============
VAGRANT BOXES
=============

REQUIREMENTS
============

* Hashicorp packer (will be installed automatically).


BUILD IMAGES
============
Build some basic plain images.

.. code-block:: Bash

  make build

Remove all builded images and cached files.

.. code-block:: Bash

  make clean
